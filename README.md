# SHIVPARIVAR FOODS & BEVERAGE

A modern website for SHIVPARIVAR FOODS & BEVERAGE, specializing in packaged drinking water.

## Project Overview

This is a responsive website showcasing two premium packaged drinking water products:
- **CLEER – Total Fresh**: Ozonized packaged drinking water with added minerals (₹20)
- **RAPID – Aqua**: Clean, safe, and refreshing drinking water (₹18)

## Features

- **Responsive Design**: Mobile-friendly layout
- **Multiple Pages**: Home, About, Products, and Contact pages
- **Product Details**: Comprehensive information including brand, category, price, and unit details
- **Professional Branding**: Consistent logo placement and color scheme

## Pages

- `index.html` - Home page with product preview
- `about.html` - Company information
- `products.html` - Product listing
- `contacts.html` - Contact information
- `cleer.html` - CLEER product details
- `rapid.html` - RAPID product details
